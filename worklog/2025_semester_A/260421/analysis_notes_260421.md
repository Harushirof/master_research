# Phase 2 制御設計ノート (2026-04-21)

## プラントモデル

```
G(s) = K / s,  K = +80.9 ± 1.7 ns/(V·s)
```

- ao1 電圧 → OCXO-B 周波数シフト → FRFR（位相差の積分）
- 純粋な積分器プラント
- 離散化: FRFR[k+1] = FRFR[k] + b * delta_u,  b = K*Ts = 24.3 ns/V

## Phase 1 の制御則（成功）

```
freq_err = (FRFR[k] - FRFR[k-1]) / Ts
u[k] = u[k-1] - Kp * freq_err
```

展開すると `u[k] = u_init - (Kp/Ts) * (FRFR[k] - FRFR[0])` → FRFR に対する比例制御。
閉ループ極: z = 1 - K*Kp ≈ 0.992（安定だがどこに収束するかは制御できない）。

## Phase 2 失敗の原因

| 試行 | 構造 | 問題 |
|------|------|------|
| 251216 v2 | u = u0 - k1*e（絶対値指令、比例のみ）| 定常偏差 = (u0-u_eq)/k1 → 78ns に収束（目標25ns）|
| 251216 v3 | v2 + wrap 処理 | wrap が誤差の単調性を破壊 + ゲイン不足で発散 |
| 260109 | 2状態FB（絶対値指令）| 偏差→飽和→制御権喪失で発散 |

共通原因: **絶対値指令には積分作用がなく、未知の平衡電圧 u_eq を補償できない。**

## Phase 2 コントローラ設計

```
e_phase  = FRFR_ref - FRFR_unwrapped     [ns]
freq_err = dFRFR/dt                       [ns/s]
delta_u  = Ki * e_phase - Kd * freq_err   [V]
u[k]     = clamp(u[k-1] + delta_u, 0, 5) [V]   ← 積分型更新
```

Phase 1 に `Ki * e_phase` を1項追加しただけ。

### ゲイン設計

設計仕様: omega_n = 0.3 rad/s, zeta = 0.8
閉ループ極: z = 0.924 ± 0.039j (|z| = 0.925)
整定時間: ≈ 15秒

| パラメータ | 値 | 役割 |
|-----------|-----|------|
| Ki | 0.0003 V/ns | FRFR を目標値に引っ張る |
| Kd | 0.0018 V/(ns/s) | ドリフトを抑制（減衰） |

### データ解析まとめ
