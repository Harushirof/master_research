# ベンチマーク位相雑音データ カタログ（260611）

> このフォルダで揃えている市販・公開の位相雑音(dBc/Hz)データの一覧。
> `excelyomi.m` は `dBCperHz.xlsx` の各シートを読み、figure1=そのまま / figure2=**10MHz換算**で重ね描きする（260611で20→10MHz化）。
> 換算式: `L_10MHz = L_f + 20·log10(0.01[GHz] / f[GHz])`（搬送波換算、kijun=0.01）。

## 3. データの偏り（追加で埋めるべき所）

- **10MHz OCXO の市販高性能機**（Wenzel, Morion MV89A 等）が手薄 → 自作の比較相手として重要。
- **GPSDO**（要旨のベンチマーク「−100 dBc/Hz級」の主役）が未収録。
- **Cs一次標準**（5071A 等）が未収録。
- 低コスト帯（MEMS, 安価TCXO）の対比サンプルも薄い。

→ 下記「3.5 追加データ」で収集・整備（Web調査43機種, 出典検証済み）。

## 4. 使い方（10MHz換算図）
```matlab
cd('...\260611\2024 0314 2.5GHz信号源重ね描き')
excelyomi          % 元のまま（figure2=20MHz換算）。本体は据え置き
excelyomi_10MHz    % 新規: figure2=10MHz換算（本命）
% figure2 を保存 → ベンチマークマップ
```
- `excelyomi_10MHz.m` … 10MHz換算版（**新規**。`excelyomi.m` は20MHzのまま据え置き）
- `overlay_benchmark_10MHz.m` … 既存xlsx14機種(10MHz)＋本研究2
- `overlay_benchmark_extra_10MHz.m` … 追加43機種(種別色)＋本研究2 （旧 full をリネーム）
- `overlay_benchmark_all_10MHz.m` … **全部入り**: 既存14(重複除外で11)＋追加43＋本研究2
  - 重複(PRS10, cybershaft×2)はデータシート側(追加)を採用しxlsx側を除外
