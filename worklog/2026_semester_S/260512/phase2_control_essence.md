# Phase 2 制御則の根幹

## 一言で

> **「位置のズレ」で押し、「速度」でブレーキ。それを前の電圧に積み増す。**

## 数式

```
Δu      = Ki · (FRFR_ref − FRFR_unwrapped)  − Kd · dFRFR/dt
u[k+1]  = clamp( u[k] + Δu , 0V, 5V )
```

## 各項の役割

| 項 | 役割 | 直感 |
|---|---|---|
| `Ki · e_phase` | **目標から遠ければアクセル増** | 距離が押す |
| `-Kd · freq_err` | **速く動いてたらアクセル戻す** | 速度で抑える |
| `u[k+1] = u[k] + Δu` | **積分型（微調整を積む）** | 「いま必要な電圧」を勘で知らなくていい |

## なぜ「積分型更新」が肝か

普通の比例制御は **絶対値で指令** する:
```
u = u0 − k · e_phase     ← u0 を勘で決める必要がある
```
→ OCXO の平衡電圧 `u_eq` は個体差・温度ドリフトで毎回違う
→ u0 を外すと永遠に定常偏差が残る

**積分型更新** なら:
```
u[k+1] = u[k] + (Ki·e − Kd·df)    ← e=0 になるまで u が動き続ける
```
→ `u_eq` を知らなくても、e=0 になった瞬間に u が「ちょうど必要な値」に落ち着く

これが **Phase 1 → Phase 2 への発展の本質**（Phase 1 は速度ゼロにするだけだったので止まる場所は不定。Phase 2 は位置誤差も積分するので狙ったところで止まる）。

## 車のアナロジー

| 制御の言葉 | 車の言葉 |
|---|---|
| FRFR | 車の **位置** |
| dFRFR/dt | 車の **速度** |
| ao0 電圧 | アクセル **踏み量** |
| K（プラントゲイン） | アクセル → 加速度 の変換係数 |
| FRFR_ref | **停めたい場所** |

> 「アクセル踏み量を変えて、ある地点で速度ゼロにして停車」 = この研究の Phase 2 と同型。

## 設計値（260421 ノートより）

| パラメータ | 値 | 役割 |
|-----------|-----|------|
| Ki | 0.0003 V/ns | 位相誤差ゲイン（目標へ引っ張る） |
| Kd | 0.0018 V/(ns/s) | 周波数誤差ゲイン（ドリフト減衰） |
| Ts | 0.3 s | サンプリング周期 |
| du_max | 0.05 V/step | レートリミット |

設計仕様: ω_n = 0.3 rad/s, ζ = 0.8, 整定時間 ≈ 15 s
