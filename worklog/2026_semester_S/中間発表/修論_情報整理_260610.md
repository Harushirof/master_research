## 1. 研究の一貫ストーリー（背骨）

```
究極目標: 位相雑音のない基準信号源
   ↓ なぜ重要か: 通信容量・測位・計測・原子時計比較が向上
ベンチマーク: 市販/GPSDO は近傍 −100 dBc/Hz 級（高性能機は大型・高価・外部基準依存）
   ↓ ギャップ: 安価なOCXO単体は限界。特に <1s（オフセット≳1Hz）の近傍位相雑音
アプローチ: 複数OCXOを同期して平均 → 無相関雑音は N台で 1/N（2台で −3 dB）
   ↓ 前提が2つ
   (a) 雑音を注入しない同期      … 制御の課題
   (b) <1s の高精度位相雑音測定  … 測定の課題
現状: (a)実測モデルFBがHOLD並みに到達 / (b)録音+ヒルベルトでps級測定を構築
   ↓
今後: 2台平均で −3dB 実証 → N拡張 → 双方向結合(1/N) → アラン分散で長期評価
```

**キーメッセージ**: 「単体を買い替えるのではなく、安価な発振器を**同期・平均**して高純度化する。そのために『雑音を注入しない同期』と『<1s高精度測定』を確立しつつある。」

## 4. 評価指標の定義（用語統一）

| 指標 | 定義 | 用途 |
|---|---|---|
| L(f) [dBc/Hz] | 片側位相雑音スペクトル密度。`L=10log10(Sφ/2)`, `Sφ=(2πf0)²·Sx` | 近傍位相雑音の主指標（搬送波 f0=10 MHz基準） |
| 時間ジッタ x(t) [s] | エッジ時刻誤差。録音ヒルベルトの直接出力（2台差 sa=x1−x2） | std[ps]で要約 |
| FRFR [ns] | オシロのエッジ間時間差。0-100ns周期ラップ | 制御用の粗い位相差 |
| アラン偏差 σ_y(τ) | 周波数安定度の時間領域指標 | 長期(>1s)評価・FB ON/OFF比較（今後） |
| 1/N低減 | 無相関N台平均で位相雑音 −10log10(N)（2台−3dB） | 平均化の理論根拠 |

> 注意: 「位相同期」(当面の手段)と「位相雑音低減」(究極目標)を混同しない。同期は平均化の**前提**であって目的ではない。

## 5. 残課題・必要な追加実験（優先順）

1. **<0.1Hz（長τ）の評価**: 今の10s窓では見えない。長時間録音/アラン偏差で「FBが長期でHOLDを超える」交差点を示す。
2. **2台平均の実証**: ミキサ等で同期2台を平均 → 単体比 −3 dB を実測（研究の核心・未実証）。Ferrand 2016 が直接の参考。
3. **分解能向上**: 1/10分圧で 1LSB 0.26→0.026 ns/s。注入ノイズ床を下げる。
4. **リアルタイムps測定**: 現状の録音はオフライン（TASCAMはUSBライブ不可と確定）。ライブ化には別途USBオーディオI/Fが必要 → リアルタイムFBへの布石。
5. **双方向（相互）結合 = Phase 3**: 1/N低減則は相反的相互結合が前提（現状は片方向＝マスタースレーブで3dB利得は出ない）。
6. **kpe0.04 異常値の裏取り**: 非単調＝過渡混入の疑い。波形確認/再窓。

## 6. 先行研究の対応づけ（要旨・本論で引く）

| 主張 | 文献 | 本研究での役割 |
|---|---|---|
| N台相互結合で位相雑音 1/N（2台−3dB） | Chang & York, IEEE T-MTT 1997 | 平均化アプローチの理論的原典。**双方向が条件**＝Phase 3の根拠 |
| ミキサで2台を平均→3dB低減 | Ferrand, IEEE SPL 2016 | 2台平均の直接手法（Phase 4参考） |
| 実験で1/N実証（N=2で3dB） | Zhang et al., PRL 2015 | 「2台で3dB」の定量先例 |
| 多数時計の合成が単体超え | NIST AT1 | 動機づけの権威ある運用例 |
| 合成は短期を犠牲に長期最適化 | Greenhall 2012 | 「FBが短期に注入」の同型知見 |
| Allan分散（評価指標） | Allan 1966 | 長期安定度評価の標準 |
