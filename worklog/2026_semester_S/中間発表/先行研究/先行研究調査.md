# 先行研究調査 — 精密周波数基準信号源と位相同期

## 1. 周波数基準信号源（オシレータ本体）

### 1.1 原子時計系（最高性能の基準）

| 種別 | 概要 | 性能（相対安定度） | 補足 |
|------|------|------|------|
| **★ セシウム原子時計（Cs）** | SI 秒の定義基準。ビーム型と噴水型 (Cs fountain) | 10⁻¹³ 〜 10⁻¹⁶ | 一次標準。国家標準研究所が保有 |
| **★ 水素メーザー (H maser)** | 水素原子の超微細遷移を利用 | 短期 10⁻¹⁵、長期 10⁻¹⁴ | 短期安定度が極めて高く、VLBI 等で使用 |
| ルビジウム原子発振器（Rb） | Rb87 の遷移 | 10⁻¹¹ 〜 10⁻¹² | 二次標準。市販品多数（民生〜業務用） |
| **★ 光格子時計** | 光遷移を用いた次世代原子時計（香取俊宏ら） | 10⁻¹⁸ 級 | SI 秒再定義候補 |
| 単イオン光時計 | 単一イオンの光遷移（NIST, PTB 等） | 10⁻¹⁸ 級 | 校正用、非常に大型 |
| チップスケール原子時計（CSAC） | Rb 系小型化（数 cc） | 10⁻¹¹ | 携帯機器・軍事用途 |

### 1.2 水晶発振器系（本研究の対象）

| 種別 | 概要 | 性能 | 補足 |
|------|------|------|------|
| XO（水晶発振器） | 標準的水晶発振器 | 10⁻⁵ | 民生用最廉価 |
| TCXO | 温度補償型 | 10⁻⁶〜10⁻⁷ | 携帯機器標準 |
| **★ OCXO（恒温槽付）** | 一定温度の恒温槽内で動作 | 10⁻⁸〜10⁻¹¹ | **本研究の対象** |
| DOCXO | 二重恒温槽 | 10⁻¹¹〜10⁻¹² | 業務用高性能 |
| BVA crystal | 仏 LNE-SYRTE 発の超高安定構造 | 10⁻¹³ 級 | 短期安定度で水素メーザーに匹敵 |

## 2. 複数発振器のアンサンブル化（本研究のアプローチに直結）

### 2.1 時刻スケールのアンサンブル合成

| 項目 | 概要 | 本研究との関係 |
|------|------|---------------|
| **★ 国際原子時 TAI（BIPM）** | 世界中 ~400 台の原子時計を統計合成して生成 | 「平均化により単体を超える性能」という本研究のアイデアの原点 |
| ALGOS アルゴリズム | TAI 生成に用いる重み付き平均アルゴリズム | アンサンブル化の理論的枠組み |
| AT1 / AT2 (NIST) | カルマンフィルタを用いた時刻スケール生成 | 統計推定によるアンサンブル時刻 |
| TA(JST) など各国時刻 | 各国標準研の独立タイムスケール | 国家基準の合成例 |

### 2.2 発振器位相同期手法

| 項目 | 概要 | 本研究との関係 |
|------|------|---------------|
| **★ Phase-Locked Loop (PLL)** | 古典的な電子的位相同期回路（VCO + 位相比較器） | 同期手法の基本原理 |
| 注入同期 (injection locking) | 強制信号の注入により発振器を引込同期させる | 受動的同期の代表例 |
| Digital PLL / ADPLL | デジタル領域での PLL | 現代の通信機器標準 |
| Master-Slave 構成 | 1台のマスターに対し複数スレーブを位相同期 | 本研究と類似（ただし平均化ではなく従属） |

## 3. 時間・周波数比較／配信技術

| 項目 | 概要 | 性能 | 補足 |
|------|------|------|------|
| **★ GPS / GNSS 同期** | 衛星信号を基準とする時刻配信 | 数 ns 〜数十 ns | GPS / GLONASS / Galileo / BeiDou |
| **★ GPS 規律発振器（GPSDO）** | OCXO/Rb を GPS で長期規律 | 短期: 発振器の性能、長期: GPS の性能 | 民生〜業務用の主流。本研究の応用方向 |
| Common-View GPS | 2 地点が同じ衛星を同時観測して時刻比較 | sub-ns | 国家標準研究所間の比較 |
| **★ 双方向衛星時刻周波数比較 (TWSTFT)** | 2 地点が静止衛星越しに信号を双方向交換 | sub-ns | TAI 生成への入力 |
| **★ 光ファイバー周波数配信** | 国研間を結ぶ高安定リンク（NICT–東大 等） | 10⁻¹⁹ 級 | 光格子時計の比較に使用 |
| White Rabbit | Ethernet ベースの sub-ns 同期プロトコル | sub-ns | CERN 等で実装 |
| PTP (IEEE 1588) | パケットベース時刻同期 | μs 〜 sub-μs | 産業用ネットワーク |

## 4. 位相ノイズ・周波数安定度の評価手法

| 項目 | 概要 | 本研究との関係 |
|------|------|---------------|
| **★ アラン分散 / アラン偏差** | 周波数安定度の時間領域指標。短期〜長期を一括評価 | 本研究の評価軸候補 |
| L(f) スペクトル密度 | 位相ノイズの周波数領域表現 | 究極目標の定量化 |
| Cross-correlation 位相ノイズ測定 | 2 系統を相関させて測定系のノイズを除去 | 高感度測定の標準手法 |
| TIE (Time Interval Error) | ジッタ評価 | 通信業界での標準 |

## 5. 位相ノイズが減ることで何が良くなるか（用途例）

| 分野 | 効果 |
|------|------|
| コヒーレント光通信 | 高次変調（64QAM 等）が可能になり、伝送容量が増大 |
| GNSS 測位 | 受信機側の位相安定度向上により測位精度が改善 |
| レーダー（特にコヒーレントレーダー） | ドップラー分解能の向上、低速ターゲットの検出 |
| 電波天文・VLBI | 干渉計の基線安定性向上、観測感度の向上 |
| 原子時計の校正・比較 | 比較精度の向上、SI 秒再定義の議論を支える |
| 量子コンピューティング | 量子ビット制御の位相精度向上 |
| 5G/6G 通信 | フェーズドアレイの位相精度向上、ビームフォーミング高度化 |
| 重力波検出（LIGO 等） | レーザー位相安定度向上、検出感度の向上 |

## 6. 本研究の位置付け（先行研究からの導出）

- **アンサンブル化により単体を超える性能を得る** という発想自体は原子時計領域で確立されている（TAI）
- しかし、**水晶発振器の小規模アンサンブル**（数台規模、研究室レベル）はあまり研究例がない
- **GPSDO** は単体 OCXO を外部基準（GPS）で規律する手法だが、外部基準への依存がある
- 本研究は **外部基準を必要とせず、内部の複数 OCXO どうしを能動的に同期させて平均化**することで、外部依存なしに位相安定度を向上させる試み
- ナノ秒〜ピコ秒オーダーでの位相同期を、フィードバック制御によって実験的に検証する

## 7. 要旨に入れる候補（★印を再掲）

要旨は短いので、以下から選んで言及する:

- 原子時計系: **セシウム** / **水素メーザー** / **光格子時計**
- アンサンブル化の例: **TAI（国際原子時）**
- 同期・配信: **GPSDO** / **TWSTFT** / **光ファイバー周波数配信** / **PLL**
- 評価指標: **アラン分散**

これらを「世の中ではこういう手法で精密信号源・同期が実現されている → ただし本研究の領域（小規模・低コスト・外部基準なし）は未開拓」という流れで配置する。
