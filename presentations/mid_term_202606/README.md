## 研究の大筋ストーリー（発表用）

1. **動機**: 2台のOCXO（水晶振動子）を位相同期できれば、多数の発振器による分散時刻基準が作れる
2. **現象**: オシロ上の FRFR（周波数比 / 位相差 [ns]）を見ることで同期状態を可視化できる
3. **Phase 1**: "任意の FRFR 値で止める"（値は制御できないが停止はする）を 10/10 の成功率で達成
4. **Phase 2**: "指定した FRFR 値で止める" に拡張中（プラント同定 → 積分型 PI 制御の設計完了）
5. **将来展望**: 2台同期 → 3台以上への拡張、ミキサーによる同期（Phase 3, 4）
